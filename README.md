# 💳 Credit Card Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Credit Card Analytics Dashboard built using Power BI. It analyzes customer demographics and transaction data to generate actionable business insights.

The solution consists of two dashboards:

- Customer Report
- Transaction Report

---

## 🎯 Objective

To design interactive dashboards that provide insights into:

- Customer segmentation
- Revenue trends
- Transaction behavior

---

## 📂 Dataset

The project uses two datasets:

- **Customer Data**: Demographics (age, gender, income, etc.)
- **Transaction Data**: Transaction details (amount, date, category)

---

## ⚙️ Tools & Technologies

- Power BI
- Power Query
- DAX (Data Analysis Expressions)

---

## 🔄 Workflow

1. Data Import & Cleaning (Power Query)
2. Data Modeling (Relationships using Client_Num)
3. Feature Engineering (DAX calculations)
4. Dashboard Development
5. Interactivity (Slicers, Filters, KPIs)

---

## 📊 Key Features

### Dashboard 1: Customer Report

- Customer segmentation (Age, Gender, Income)
- Revenue contribution analysis
- KPI cards

### Dashboard 2: Transaction Report

- Weekly revenue trends
- Transaction volume analysis
- Category-wise spending insights

---

## 📈 Key Insights

- Total Customers: ~10K+
- Total Revenue: ~3M
- Weekly Growth: 10.15%
- Female customers contribute more revenue than males
- Medium-income group dominates customer base
- Blue card category generates majority of revenue

---

## ⚠️ Challenges

- Data cleaning and formatting issues
- Complex DAX calculations
- Maintaining dashboard clarity with interactivity

---

## 🚀 Outcome

The dashboard enables:

- Data-driven decision making
- Identification of high-value customers
- Monitoring of transaction trends
